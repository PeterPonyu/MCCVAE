About MCC

MCC (MoCo Coupling) combines Momentum Contrastive learning with an information bottleneck within a VAE framework. It uses momentum-updated encoders and memory queues for contrastive representation learning, and a coupling layer that compresses latent codes through a low-dimensional bottleneck.

The framework supports scRNA-seq and scATAC-seq data with multiple reconstruction loss modes (MSE, NB, ZINB).
