What this surface is

A thin public landing page for the MCCVAE project and its local workflow.

What stays local

How to use it

Run the application locally with the documented startup flow and backend service.

The operational MCCVAE training interface is intentionally not exposed here as a public app. It depends on local services and a local backend contract, so this landing page keeps the public graph accurate while preserving a path to the repo and the wider portal.
